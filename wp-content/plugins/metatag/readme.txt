MetaTag is a simple to use WordPress plugin to enhance on page SEO of your website.

== Description ==

Yet another WordPress SEO plugin to enhance on page SEO of your website.

It provides a simple to use interface to enter your meta tags like meta descriptions and keywords.
The tags will automatically inserted into your website header while a visitor or search engine bot visit your website. This will enhance your presence to search engines and will work with you to increase your visitors.



**Upcoming features:**
* Automatic meta tag generator
* A nice Admin panel
* Support for custom post type


If you have further query, [contact me](http://en.ibabar.com/contact-us "Mail me directly from my blog")


== Installation ==

Upload the MetaTag plugin to your blog, Activate it, then use it while editing your post or page.

1, 2, 3: You're done!
